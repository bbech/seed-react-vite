# seed-react-vite
Starter Project with React,  Vite,  Axios , Inversify,  Openapi Generator

## Start Mock server 

```bash
$ docker-compose up
```

## Install, Genrate Client API & Run App

Install

```bash
$ npm i
```

Generate Client From Open API

```bash
$ npm run generate:api
```

Run The APP
```bash
$ npm run dev
```
